# 📖 傻瓜式部署指南（跟着做就行！）

## 第一步：安装 Git（如果还没装）

1. 访问 https://git-scm.com/download/win
2. 下载安装包，一路点击「Next」安装即可
3. 安装完成后，**重新打开**终端

---

## 第二步：创建 GitHub 仓库

1. 打开 https://github.com 并登录（没有账号先注册）
2. 点击右上角的 **+** 号 → 选 **New repository**
3. 仓库名称填：`daily-paper-tracker`（或你喜欢的名字）
4. 选择 **Private**（推荐，别人看不到）
5. **不要**勾选任何选项（不要选 README、.gitignore 等）
6. 点击 **Create repository**
7. **重要：** 把页面上显示的仓库地址复制下来（类似 `https://github.com/你的用户名/daily-paper-tracker.git`）

---

## 第三步：获取 QQ 邮箱授权码

1. 打开 https://mail.qq.com 登录
2. 点击顶部的 **设置** → **账户**
3. 向下滚动找到 **POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务**
4. 点击 **开启** POP3/SMTP 服务
5. 按提示用手机发送短信验证
6. 验证成功后会显示一个**授权码**（类似 `abcd1234efgh5678`）
7. **把这个授权码复制保存好！**（这不是你的 QQ 登录密码！）

---

## 第四步：推送代码到 GitHub

在当前目录打开终端，依次执行以下命令（一行一行执行）：

```bash
# 1. 初始化 Git
git init

# 2. 添加所有文件
git add .

# 3. 提交
git commit -m "Initial commit"

# 4. 关联远程仓库（把下面的地址换成你第二步复制的地址！）
git remote add origin https://github.com/你的用户名/你的仓库名.git

# 5. 推送到 GitHub
git branch -M main
git push -u origin main
```

执行完后，去 GitHub 仓库页面刷新一下，应该能看到代码了！

---

## 第五步：配置 GitHub Secrets（最重要！）

1. 回到你的 GitHub 仓库页面
2. 点击顶部的 **Settings**（设置）
3. 在左侧菜单找到 **Secrets and variables** → 点击 **Actions**
4. 点击绿色的 **New repository secret** 按钮

### 添加第一个 Secret：
- **Name:** `QQ_EMAIL`
- **Value:** `2026204614@qq.com`
- 点击 **Add secret**

### 添加第二个 Secret：
- **Name:** `QQ_PASSWORD`
- **Value:** （粘贴你第三步获取的授权码）
- 点击 **Add secret**

---

## 第六步：测试运行

1. 点击仓库顶部的 **Actions** 标签
2. 左侧选择 **Daily Paper Tracker**
3. 点击右侧的 **Run workflow** 按钮
4. 在弹出的窗口中点击绿色的 **Run workflow**
5. 等待 2-3 分钟，看是否显示绿色的 ✅
6. 检查你的 QQ 邮箱（包括垃圾箱）是否收到邮件

---

## 第七步：完成！

如果测试成功，那么从明天开始，**每天早上 8 点**你就会自动收到邮件啦！🎉

---

## 遇到问题？

- 如果 Git 命令报错，确认 Git 已经正确安装
- 如果邮件没收到，检查 GitHub Actions 的日志看错误信息
- 慢慢来，每一步确认成功了再进行下一步

祝你好运！🚀

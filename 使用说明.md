# 每日论文和AI进展追踪系统

## 功能介绍

这个系统会自动：
1. 从 arXiv 等渠道获取最新的 AI 和 Agent 相关论文
2. 生成包含论文摘要的总结文档
3. 下载论文 PDF 文件
4. 通过邮件发送到你的 QQ 邮箱

## 🚀 快速开始（推荐方案：GitHub Actions）

我们已经配置好 GitHub Actions 方案，**无需你的电脑一直开机**！

### 详细步骤请查看：[GitHub设置指南.md](file:///d:\Trae\TRAE SOLO CN\Project\每日论文讲解\GitHub设置指南.md)

### 简要流程：

1. **创建 GitHub 仓库** - 在 GitHub 上新建一个仓库
2. **推送代码** - 将本项目推送到 GitHub
3. **配置 Secrets** - 在 GitHub 仓库设置中添加 QQ 邮箱和授权码
4. **享受服务** - 每天早上 8 点自动收到邮件！

## 💻 本地测试（可选）

如果你想在本地先测试一下：

### 1. 获取 QQ 邮箱授权码

1. 登录 QQ 邮箱网页版
2. 进入「设置」→「账户」
3. 找到「POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务」
4. 开启「SMTP」服务
5. 生成授权码（保存好这个密码）

### 2. 创建 .env 文件

复制 `.env.example` 为 `.env`，然后填入你的信息：

```
QQ_EMAIL=2026204614@qq.com
QQ_PASSWORD=你的授权码
```

### 3. 安装依赖

```bash
pip install -r requirements.txt
```

### 4. 测试运行

```bash
python paper_tracker.py
```
